## 6. Partnership 페이지

### 소개 문구
```
주인장은 다양한 형태에 따른 파트너십을 제공하고 있습니다.
파트너의 상황에 따라 다양한 솔루션을 제공하고 있으며,
주인장이 개발하는 브랜드와 함께 최적의 솔루션을 제공합니다.
```

### 파트너십 유형

#### 1. 개발투자
```
주인장에서 진행하는 사업건에 직/간접적으로 투자를 하는 형태로서,
자본은 있지만 개발 및 운영할 여력이 되지 않은분이나,
주인장이 개발하는 사업에 소액투자하여 수익쉐어를 받고 싶은 분을
대상으로 진행하고 있습니다.

현재는 주인장의 첫 브랜드인 글램독의 숙박시설 7개, F&B시설 2개
총 9개의 개발투자가 진행되었습니다.
```

#### 2. 위탁운영
```
기존의 운영중인 시설을 다양한 이유로 직접 운영하기 어려울 경우,
주인장에게 전문적으로 경영/관리/운영을 맡기고,
이에대한 수익을 배분하는 형태입니다.

현재는 주인장의 첫 브랜드인 글램독 브랜드의 위탁운영을 진행하고 있습니다.
```

#### 3. 브랜드 가맹
```
기존의 운영중인 시설을 직접 운영하면서 주인장의 브랜드를 사용하고
이에대한 서비스를 제공받고 일정의 수수료를 정산하는 방식입니다.

현재는 주인장의 첫 브랜드인 글램독의 브랜드 가맹을 준비중입니다.
```

### CTA 섹션
```
주인장 파트너십과 관련된 궁금하신 점이나
파트너십과 관련하여 상담이 필요하시면
언제든지 문의주세요.
```
